陳昇瑋生平

陳昇瑋博士為台灣人工智慧學校執行長、玉山金控暨玉山銀行科技長兼數位長、財團法人人工智慧科技基金會董事長、財團法人科技生態發展公益基金會執行長，曾任中央研究院資訊科學研究所研究員暨資料洞察實驗室主持人。研究領域為人工智慧、金融科技、多媒體系統、社群網路及計算社會科學等，長年推動人工智慧在製造、金融、零售、服務各產業的深化應用及數位轉型。

陳博士被中央研究院院長稱為「百年難得一見的才子」，畢生志願為「以電腦科學家的角色最大化對於社會的貢獻與影響」，透過創辦台灣人工智慧學校及台灣資料科學年會，希望讓這些由資料、數據而生的技術不再是口號，而是企業真實拿來解決問題及創造價值的工具，達到「讓『找不到人才』不再成為產官學研各界發展人工智慧的障礙，建立『自己的問題自己解決』的文化，打破被技術殖民的慣性，促成產業轉型，建立社會信心」的目標。

短暫而燦爛的生命旅程中，陳博士跨界、突破框架的思維，以及無畏的膽試與魄力，屢屢為台灣學術界及產業開創新局，非僅學術研究成果豐碩，更成為台灣產業AI化的先驅與代表人物。重要的事情沒人做就「我們來做」的精神、永遠不劃地自限的精神已成典範。

一、創辦台灣人工智慧學校

2018年初，結合學術與產業界力量，在未申請政府經費補助的前提下，兩年內培育6000位產業AI 化精英人才，成為台灣最大的人工智慧社群，並成立校友會，持續在不同產業擴大影響力。以「培育人才」做為切入點，一方面快速突圍台灣產業困境，應用人工智慧新科技接軌國際趨勢、協助台灣企業開創新局；一方面引領嶄新經營思維，做為新世代產業的基礎建設。

二、創辦台灣資料科學年會及台灣人工智慧年會

台灣首見大規模技術講座，突破台灣長期專注於硬體代工的思維，也告別大拜拜式的論壇，陳昇瑋以個人之力率領研究團隊，集結企業力量，共同改變台灣長期處於被國際科技大廠「技術殖民」的困境。從2014至2017年四屆年會中，共有來自產業與學術界的5000位資料科學專業人才、232位國內外頂尖專家參與，其中包括 AlphaGo 研發團隊負責人黃士傑等重要講者，讓大數據、人工智慧不再是人云亦云的玄學，而成為產業突破既有格局的動力。

三、身體力行協助產業發展的實踐者

以資料科學家身分進入企業，成為台灣首位金控科技長，為台灣金融科技創新指引未來方向，並為企業與學研界如中研院、台大、交大等多所學術單位建立直接合作關係。

四、創新與大眾溝通模式推動科普教育

主動投入協助中研院成立《研之有物》科普網站，為台灣學研單位首創，甫成立一年即獲得財團法人公共關係基金會「政府傳播類公關優異獎」。並受《天下雜誌》及《經濟日報》邀請定期撰寫專欄，與社會大眾溝通人工智慧技術及應用、產業數位轉型等台灣企業迫切面對的問題與解法。2019年並出版《人工智慧在台灣：產業轉型的契機與挑戰》一書，為第一本從台灣產業角度出發的人工智慧專書，上市當天即二刷，深受各界肯定。同時經常受邀至企業、公部門及法人單位演講，光是民國107年即演講一百餘場。

五、從資訊技術出發推動社會公益

積極與資訊科技相關社群連結，包括IT技術社群，如Hacks in Taiwan, SITCON, PyCon, COSCUP；科普社群如泛科學、故事等；公民科技社群如g0v、開放文化基金會、資料英雄計畫以及LASS等；還有資料科學/開放社群Open Data 聯盟、Taiwan R User Group等，藉此延伸學術研究者對社會的關懷。
